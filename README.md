# مطبعة العنقاء - موقع إلكتروني

موقع إلكتروني احترافي لمطبعة العنقاء يتضمن عدة صفحات وخدمات.

## الصفحات المتوفرة

1. **الصفحة الرئيسية** (`index.html`) - عرض أعمال المطبعة وخدماتها
2. **صفحة الأسعار** (`prices.html`) - عرض أنواع الورق والأسعار:
   - الورق العادي (3 متر)
   - الورق الملون (5 متر)
   - الورق المطبوع (13 متر)
3. **صفحة الاتصال** (`contact.html`) - معلومات الاتصال ونموذج التواصل
4. **صفحة الأخطاء** (`errors.html`) - صفحة 404 للأخطاء
5. **صفحة الطلب** (`request.html`) - نموذج لإرسال طلبات الطباعة عبر تيليجرام

## المميزات

- تصميم عصري وجذاب
- متجاوب مع جميع الأجهزة (موبايل، تابلت، كمبيوتر)
- دعم اللغة العربية (RTL)
- تكامل مع بوت تيليجرام لإرسال الطلبات
- واجهة مستخدم سهلة الاستخدام

## التثبيت والتشغيل

### المتطلبات

- Node.js (الإصدار 14 أو أحدث)
- npm (يأتي مع Node.js)

### خطوات التثبيت

1. تثبيت المكتبات المطلوبة:
```bash
npm install
```

2. تشغيل الخادم:
```bash
npm start
```

أو للتطوير مع إعادة التشغيل التلقائي:
```bash
npm run dev
```

3. افتح المتصفح وانتقل إلى:
```
http://localhost:3000
```

## إعداد بوت تيليجرام

1. تأكد من أن لديك بوت تيليجرام برقم الرمز المميز (Token)
2. أرسل رسالة إلى البوت على تيليجرام (أي رسالة)
3. يمكنك الحصول على معرف المحادثة (Chat ID) من:
   ```
   https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates
   ```
4. استبدل `YOUR_BOT_TOKEN` برمز البوت الخاص بك
